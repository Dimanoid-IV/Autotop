### Файлы, которые НЕ должны быть в репозитории:

- `.env` - файл с переменными окружения (уже в `.gitignore`)
- Токены и API ключи
- Пароли и секретные ключи

### Если нужно добавить конфигурацию:

Создайте файл `.env.example` с примером структуры (без реальных значений):

```env
DATABASE_URL="your_database_url_here"
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your_secret_here
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

## 🔄 Следующие шаги

1. **Клонирование репозитория**:
   ```bash
   git clone https://github.com/Dimanoid-IV/Autotop.git
   cd Autotop
   npm install
   ```

2. **Настройка окружения**:
   - Создайте файл `.env` (см. `ENV_SETUP.md`)
   - Настройте базу данных (см. `DATABASE_SETUP.md`)

3. **Запуск проекта**:
   ```bash
   npm run dev
   ```
