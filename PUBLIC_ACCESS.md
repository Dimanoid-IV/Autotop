# Публичный доступ для незарегистрированных пользователей

## Статус

✅ **Все API endpoints для просмотра карточек фирм доступны незарегистрированным пользователям**

## Доступные endpoints

### Просмотр карточек фирм

1. **GET `/api/businesses`** - Список всех фирм
   - Параметры: `page`, `limit`, `city`, `category`, `search`
   - ✅ Публичный доступ (без авторизации)

2. **GET `/api/businesses/[id]`** - Детали конкретной фирмы
   - ✅ Публичный доступ (без авторизации)

3. **GET `/api/top-businesses`** - Топ-10 фирм
   - Параметры: `city`
   - ✅ Публичный доступ (без авторизации)

4. **GET `/api/latest-reviews`** - Последние отзывы
   - Параметры: `limit`
   - ✅ Публичный доступ (без авторизации)

### Вспомогательные endpoints

5. **GET `/api/cities`** - Список городов
   - ✅ Публичный доступ (без авторизации)

6. **GET `/api/categories`** - Список категорий
   - ✅ Публичный доступ (без авторизации)

## Что требует авторизации

- **POST `/api/businesses`** - Добавление новой фирмы (требует авторизации)
- **POST `/api/reviews`** - Добавление отзыва (требует авторизации)
- **POST `/api/replies`** - Ответ на отзыв (требует авторизации)
- **GET `/api/reviews/[id]/approve`** - Одобрение отзыва (требует админ-права)
- **GET `/api/reviews/[id]/reject`** - Отклонение отзыва (требует админ-права)

## Реализация

В файле `app/api/businesses/route.ts`:
- GET метод: не использует `getServerSession`, доступен всем
- POST метод: использует динамический импорт `getServerSession` только когда нужен
