# Перезапуск MCP серверов

## 🔄 Способы перезапуска MCP серверов в Cursor

### Способ 1: Перезапуск через Command Palette (рекомендуется)

1. Откройте Command Palette: `Ctrl+Shift+P` (Windows/Linux) или `Cmd+Shift+P` (Mac)
2. Введите: `MCP: Restart Servers` или `MCP: Restart All Servers`
3. Нажмите Enter

### Способ 2: Перезапуск Cursor IDE

1. Закройте Cursor IDE полностью
2. Откройте Cursor IDE заново
3. MCP серверы автоматически перезапустятся

### Способ 3: Через настройки

1. Откройте настройки: `Ctrl+,` (Windows/Linux) или `Cmd+,` (Mac)
2. Найдите "MCP" в поиске
3. Найдите опцию "Restart Servers" или аналогичную

## 🔍 Проверка статуса серверов

После перезапуска проверьте статус серверов:
- Откройте Command Palette: `Ctrl+Shift+P`
- Введите: `MCP: Show Server Status` или аналогичную команду
- Проверьте, что все серверы запущены

## ⚠️ Если серверы не запускаются

1. Проверьте файл конфигурации `mcp.json` на ошибки
2. Убедитесь, что все необходимые переменные окружения установлены
3. Проверьте логи Cursor в Developer Console: `Help → Toggle Developer Tools`
4. Перезапустите Cursor IDE полностью

## 📝 Примечание

MCP серверы управляются самим Cursor IDE и обычно перезапускаются автоматически при необходимости. Ручной перезапуск может потребоваться после изменений в конфигурации.
